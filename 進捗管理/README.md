# Flowlog

Backlog 風の見た目と使い勝手を意識した、シンプルな進捗管理ツールです。

## できること

- プロジェクトの追加
- 課題の追加、編集
- ステータス変更
- 担当者、優先度、状態、キーワードでの絞り込み
- ローカルストレージへの保存

## 使い方

`index.html` をブラウザで開くと利用できます。

依存なしで動作する構成なので、そのままファイルを開くだけでも問題ありません。

## スマホ対応と無料公開

このツールはスマホ表示向けの調整と PWA 対応を入れてあります。`manifest.json` と `service-worker.js` を含めているので、公開後はスマホのブラウザから開いてホーム画面に追加できます。

### 無料で公開する方法

1. GitHub にこのフォルダ一式をアップロードします。
2. GitHub Pages を有効化して公開します。
3. 公開URLをスマホで開きます。

### PC とスマホで同じデータを使う方法

1. Firebase で無料プロジェクトを作成します。
2. Firestore を有効化します。
3. `firebase-config.example.js` を参考にして `firebase-config.js` を書き換えます。
4. 同じ公開URLをPCとスマホの両方で開きます。

初期状態では `firebase-config.js` は無効化してあるため、ローカル保存のまま動きます。
